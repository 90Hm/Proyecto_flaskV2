Acerca de este proyecto

Este proyecto fue creado como práctica para aprender y desarrollarnos mas con Flask y Jinja2.

Se aplicó herencia de plantillas para poder reutilizar el código común (como el encabezado, la barra de navegación y el pie de página) en todas las páginas del sitio, evitando la repetición de código y facilitando el mantenimiento.

También se implementaron estilos CSS para mejorar la apariencia del sitio, usando colres suaves, degradados y sombras para que se vea más amigable y ordenado.

La estructura del proyecto está organizada en carpetas: templates para los archivos HTML, static para CSS y otros recursos, y un archivo app.py que contiene la lógica de la aplicación.

Este proyecto sirve como base para futuras prácticas de Flask.
